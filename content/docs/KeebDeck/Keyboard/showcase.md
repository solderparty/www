Here's a showcase of some of the community projects already using the KeebDeck Keyboard, arranged from latest to oldest.

## Hackaday Superconference 2025 Badge

<a href="https://raw.githubusercontent.com/Hack-a-Day/2025-Communicator_Badge/refs/heads/main/images/2025_badge_front_supercon_logo.png"><img src="https://raw.githubusercontent.com/Hack-a-Day/2025-Communicator_Badge/refs/heads/main/images/2025_badge_front_supercon_logo.png" style="height: 500px"></a>

The Badge for the 2025 Hackaday Superconference. It uses a variation of the KeebDeck Keyboard with modified layout, and dimensions.

Learn more about it from the [announcement post](https://hackaday.com/2025/10/23/announcing-the-2025-hackaday-superconference-communicator-badge).

## WHY2025 Badge

<a href="/keeb/why2025_badge.jpg"><img src="/keeb/why2025_badge_small.jpg" style="height: 500px"></a>

The badge for the WHY2025 outdoor hacker camp. It uses the regular KeebDeck Keyboard form-factor, but with a custom logo on the Meta key.

Learn more about it from their [blog post here](https://why2025.org/post/697).

## Tanmatsu

<a href="/keeb/tanmatsu.png"><img src="/keeb/tanmatsu.png" style="height: 500px"></a>

Tanmatsu is a terminal device for hackers, makers, and tech enthusiasts. It's based around the powerful ESP32-P4 microcontroller. It uses the regular KeebDeck Keyboard form-factor, but with a few customizations to the labels.

Learn more about it from [the project's website](https://nicolaielectronics.nl/tanmatsu/).

## Fri3d 2024 Badge Communicator Add-on

<a href="/keeb/fri3d_badge.jpg"><img src="/keeb/fri3d_badge_small.jpg" style="height: 500px"></a>

An add-on for the Fri3d Camp 2024 badge. Fri3d is a family-friendly summer camp for hackers, makers, and DIY enthusiasts. It uses the regular KeebDeck Keyboard.

Learn more about it from [the add-on's documentation](https://fri3dcamp.github.io/badge_2024/en/communicator).
